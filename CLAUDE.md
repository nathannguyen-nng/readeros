# Claude Guide for readerOS

This file is intentionally short. It exists for Claude Code compatibility and
should stay small enough to be useful in every session.

Read `AGENTS.md` first. It is the shared agent index for Claude, Codex, and
other coding agents working on this repository.

For task-specific workflows, use the same repo skill folders under
`.agents/skills/` as checklists, then read only the linked `agent-docs/`
reference files that apply to the current change.

Do not turn this file back into a full project manual. Add shared always-on
rules to `AGENTS.md`, reusable task workflows to `.agents/skills/`, and detailed
reference notes to `agent-docs/`.
